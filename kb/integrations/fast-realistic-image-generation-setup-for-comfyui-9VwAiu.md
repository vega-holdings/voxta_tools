# Fast Realistic Image Generation Setup for ComfyUI

> Recommended ComfyUI model and LoRA settings for fast, realistic image generation.

## Answer

For fast (approx. 3 seconds) and realistic image generation in ComfyUI, the following configuration is recommended:

*   **Workflow**: Default ComfyUI workflow with a LoRA stack (available in templates).
*   **Model**: `illustriousRealismBy_v10VAE.safetensors`
*   **LoRAs**:
    *   `Ass Size Slider - Illustrious.safetensors`
    *   `ILXL_Realism_Slider_V.1.safetensors`
