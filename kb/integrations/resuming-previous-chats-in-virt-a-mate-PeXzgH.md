# Resuming Previous Chats in Virt-a-Mate

> Select the previous session from the dropdown menu in the VAM plugin to resume a chat.

## Answer

To resume a previous conversation in Virt-a-Mate:
1. Locate the dropdown menu in the Voxta plugin interface designed for selecting chat sessions.
2. Select the specific previous chat you wish to continue.
3. Activate the chat.
