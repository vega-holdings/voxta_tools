# Stable Diffusion Backend and Sampler Recommendations

> Comparison of Automatic 1111 vs Forge and recommended samplers for different art styles.

## Answer

When choosing an image generation backend for Voxta:

**Backend Choice**
*   **Automatic 1111**: Recommended for beginners requiring easier direct localhost connection.
*   **Forge**: Recommended for ready-made presets (negative prompts, styles) and native support for Flux models, though host setup may be more complex.

**Sampler Recommendations**
*   **Anime/2D**: DPM++2M Karras
*   **Realism**: Euler A
*   **Flux Models**: Euler
