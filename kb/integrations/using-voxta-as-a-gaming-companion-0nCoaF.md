# Using Voxta as a Gaming Companion

> Voxta can serve as a voice companion for games like Elite Dangerous using local LLMs.

## Answer

Voxta can be utilized as a voice companion for immersive gaming experiences in titles like *Elite Dangerous* and *No Man's Sky*. Users report running local LLM backends (such as **LM Studio** or **KoboldCpp**) alongside Voxta to enable voice interactions while playing.
