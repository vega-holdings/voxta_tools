# Civitai Image Generation Credits and NSFW

> Civitai NSFW image generation requires credits which can be earned or bought.

## Answer

Civitai integration allows for NSFW image generation, though it may experience delays. NSFW generation typically requires credits (Buzz), which can be purchased or earned through Civitai's moderation system (Knight's Order).
