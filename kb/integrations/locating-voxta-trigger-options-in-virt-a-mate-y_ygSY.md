# Locating Voxta Trigger Options in Virt-A-Mate

> How to navigate the VaM UI to find Voxta trigger options.

## Answer

To find available triggerable options for Voxta within Virt-A-Mate (VaM):

1. Go to the **Trigger** menu.
2. Select the **Atom** containing Voxta.
3. Select the **Plugin**.
4. You will see all available triggerable options listed.
