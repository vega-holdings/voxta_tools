# Configuring and Running the Voxta C# Sample Provider

> How to configure and run the Voxta Sample Provider App.

## Answer

To customize the behavior of the C# sample provider (such as setting the time/interval for auto-replies), edit the `AutoReplyProvider.cs` file. To start the provider, execute `Voxta.SampleProviderApp.exe`.
