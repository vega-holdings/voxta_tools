# Virt-A-Mate License Recommendation

> Recommends the Creator license for Virt-A-Mate to allow asset modification.

## Answer

When using Voxta with Virt-A-Mate, it is recommended to have the **Creator** license for Virt-A-Mate. This license enables the ability to download and modify assets, which is beneficial for setting up scenes and characters effectively.
