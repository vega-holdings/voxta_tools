# Installing ExLlamaV2 Backend

> Steps to install ExLlamaV2 via Manage Services and locate it in the UI.

## Answer

To set up ExLlamaV2, navigate to the **Manage Services** section in the Voxta App and locate the installer for ExLlamaV2. Once installed, the service can be monitored and selected via the Quick Access menu (the slideout panel on the right side of the interface).
