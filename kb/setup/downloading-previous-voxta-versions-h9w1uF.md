# Downloading Previous Voxta Versions

> Instructions on how to find and download older versions of Voxta from the portal.

## Answer

You can download older versions of Voxta by visiting the [Voxta Portal](https://portal.voxta.ai) download page and clicking the **Show previous releases** option.
