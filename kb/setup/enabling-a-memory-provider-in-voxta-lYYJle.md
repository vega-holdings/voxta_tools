# Enabling a Memory Provider in Voxta

> Steps to install and activate a memory provider via the Voxta interface.

## Answer

To enable a memory provider (such as ChromaDB) in Voxta:
1. Navigate to **Manage Services** (or Manage Modules).
2. Install the desired memory module.
3. Open the **Settings** menu (top right cogwheel icon).
4. Select the installed module under the **Memory Provider** dropdown.
