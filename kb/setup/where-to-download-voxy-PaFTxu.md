# Voxy Download Status and Distribution

> Voxy is currently unreleased and unavailable for download. Once available, it will be distributed via the #voxy-releases Discord channel, not the cloud portal.

## Answer

### Current Availability

Voxy has not been released yet and is currently unavailable for download. This applies to all users regardless of subscription tier or Patreon status.

### Where to Find Downloads

Once Voxy becomes available, downloads will **not** be located in the cloud portal. Instead, they will be distributed via the official Discord server in the `#voxy-releases` channel.
