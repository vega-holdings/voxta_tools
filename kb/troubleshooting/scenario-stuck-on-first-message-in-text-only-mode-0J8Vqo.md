# Scenario Stuck on First Message in Text-Only Mode

> Restarting the app fixes an issue where scenarios get stuck in text-only mode.

## Answer

This issue, where the dialog loops on the first message during text-only scenario testing, can often be resolved by restarting the Voxta application.
