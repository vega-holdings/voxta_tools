# Fix for 'Preset cannot be tracked' error in Voxta v140

> Workarounds for the 'Preset cannot be tracked' error in Voxta v140 when adding Text Generation Web UI.

## Answer

This error is a known issue in Voxta v140 when adding the Text Generation Web UI service.

**Workarounds:**
1. Use the **OpenAI Compatible** service type to connect to Text Generation Web UI (Oobabooga) instead of the dedicated service option.
2. Downgrade to Voxta v139.
3. Wait for a patch addressing this issue in v140.
