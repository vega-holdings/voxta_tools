# Troubleshooting Vision Service Installation

> Steps to troubleshoot Vision service by reinstalling components and checking Windows SDK/FlashCap.

## Answer

If you are experiencing issues with the Vision service, try the following steps:

1. Uninstall and reinstall all Vision-related components.
2. Ensure the **Windows SDK** and **FlashCap** modules are set up.
3. Click **Install all services dependencies** located at the top left of the module list window.
