# Fixing Audio Clipping in WAV Files

> Use Audacity's Clip Fix effect to repair clipping in loud WAV files.

## Answer

If WAV files are too loud and contain clipping (distortion/spikes), the **Clip Fix** effect in Audacity can be used to remove the spikes and repair the audio.
