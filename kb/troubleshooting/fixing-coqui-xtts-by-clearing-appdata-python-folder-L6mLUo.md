# Fixing Coqui XTTS by clearing AppData Python folder

> Delete the python312 folder in AppData/Roaming and reinstall Coqui XTTS to fix issues.

## Answer

If you are experiencing issues with Coqui XTTS (potentially related to Python path conflicts), try deleting the `python312` folder located in your `%AppData%\Roaming` directory. After deleting the folder, reinstall Coqui XTTS.
