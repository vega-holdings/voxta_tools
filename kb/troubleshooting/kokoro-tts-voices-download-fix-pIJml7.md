# Kokoro TTS Voices Download Fix

> Manually download voices.bin from the Kokoro ONNX GitHub releases if the default URL returns 404.

## Answer

If the default download URL returns a 404 error, the voices file location may have changed. You can manually download the `voices.bin` file directly from the [Kokoro ONNX releases page](https://github.com/thewh1teagle/kokoro-onnx/releases/).
