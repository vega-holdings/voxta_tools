# Troubleshooting Generic Character Chat Errors

> Use F2 to check logs when encountering generic errors with characters.

## Answer

To diagnose generic errors affecting all characters, check the application logs. Press **F2** to open the logs and look for error messages or explanations that appear earlier in the log history.
