# Error Resuming Chats with Images in Voxta 1.1.1

> Resuming chats with generated images fails in v1.1.1 due to an unsupported file type error; a fix is pending in the next build.

## Answer

This is a known bug in Voxta release 1.1.1. It occurs when trying to load or resume a chat history that contains images generated via the 'Imagine' feature.

**Status:**
The issue has been identified and fixed by the developers, but the fix requires a new build release. Users will need to update to the version following 1.1.1 once it is available to resolve this error.
