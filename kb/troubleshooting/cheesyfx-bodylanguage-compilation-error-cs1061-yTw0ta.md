# CheesyFX BodyLanguage Compilation Error (CS1061)

> Fix for CS1061 compilation error in CheesyFX BodyLanguage by updating the plugin.

## Answer

This error typically indicates an outdated version of the `CheesyFX BodyLanguage` plugin or a mismatch with dependencies. To resolve this:

1. Verify that `CheesyFX BodyLanguage` is present in your addon folder.
2. Update the plugin to the latest version (e.g., version .71 or newer).

This error often appears alongside missing atom errors (e.g., `Could not find atom by uid UIButton...`) because the script failure prevents the scene from initializing correctly.
