# Memory Book Date Display Error

> Incorrect dates or <date> placeholders in Memory Book are a known bug fixed in the next build.

## Answer

This is a known bug where dates in the Memory Book display incorrectly or as `<date>`. The issue has been identified and is scheduled to be fixed in the next software build.
