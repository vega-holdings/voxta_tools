# Repeated Messages with Vision in Voxy Beta.23

> Fix for repeated messages bug in Voxy beta.23 when using Vision.

## Answer

Users on Voxy beta.23 may experience multiple repeated user messages when the vision action is prompted in the chatbox. This is a known bug in that specific version. A fix has been released and can be downloaded from the [Voxta Portal](https://portal.voxta.ai).
