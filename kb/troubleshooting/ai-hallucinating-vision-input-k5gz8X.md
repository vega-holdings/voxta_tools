# AI Hallucinating Vision Input

> Ensure Windows SDK Augments and a compatible Chat Mode are active if the AI is hallucinating vision.

## Answer

This issue typically occurs when the vision system is not correctly engaged, causing the LLM to hallucinate visual context. To fix this:

1. Ensure **Windows SDK Augments** are configured/installed.
2. Verify that the selected **Chat Mode** supports augments (vision capabilities).
