# Scenario Import Fails (Already Exists)

> Enable overwrite when importing existing scenarios.

## Answer

If the logs indicate "scenario already exist and overwrite is not enabled," the import is blocked to prevent accidental data loss. You must enable the **Overwrite** option in the import settings before attempting to import the JSON again.
