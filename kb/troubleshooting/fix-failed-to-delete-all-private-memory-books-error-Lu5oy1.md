# Fix: Failed to delete all private memory books error

> Workaround for the 'Failed to delete all private memory books' error by cloning the character and manually deleting the original files.

## Answer

This error occurs when Voxta struggles to clean up a large number of private memory books associated with a character. 

**Steps to resolve:**
1. **Update Voxta:** Ensure you are on the latest version, as patches have been released to address this specific issue.
2. **Workaround (if update fails):**
   - **Clone** the affected character within the Voxta UI to preserve settings.
   - Navigate to your Voxta installation folder via File Explorer.
   - **Manually delete** the original character files/cards directly from the folder.
   - Continue using the cloned version of the character.
