# Accessing Interface when running Voxta Server directly

> Explains how to access the web interface when running the server executable directly.

## Answer

`Voxta.Server.exe` runs the server process without a built-in GUI window. To access the interface:

1. Open a web browser.
2. Navigate to `http://127.0.0.1:5384/`.

Alternatively, running `Voxta.DesktopApp.exe` will launch the server and open the interface automatically.
