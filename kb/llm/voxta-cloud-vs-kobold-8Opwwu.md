# Voxta Cloud vs Kobold

> Voxta Cloud is a paid remote service; Kobold is a free local service.

## Answer

**Voxta Cloud** is a paid service included with Voxta Patreon subscriptions that handles AI processing remotely using a credit system. It is an all-in-one solution for online services.

**Kobold** is a free, locally hosted backend that runs Large Language Models (LLMs) on your own computer's hardware.
