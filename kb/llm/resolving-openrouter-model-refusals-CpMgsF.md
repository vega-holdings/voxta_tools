# Resolving OpenRouter Model Refusals

> Switch to a different OpenRouter model if the current one refuses to answer.

## Answer

If you experience the AI refusing to answer prompts while using OpenRouter, it is likely due to the specific model's safety filters or alignment. Switching to a different model within OpenRouter usually resolves the issue.
