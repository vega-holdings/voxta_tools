# Changing LLM Model in Voxta Cloud

> Instructions on locating the LLM selection settings in Voxta Cloud.

## Answer

To change the LLM model in Voxta Cloud, navigate to the **Text Generation** menu.

**Requirements:**
- An active Patreon subscription.
- Your Patreon account must be linked to your Discord account to access Voxta Cloud features.
