# Supported File Formats for Animated 2D Assets

> aPNG and WebP are supported formats for animated 2D assets.

## Answer

No, for animated pictures you can also use `aPNG` (Animated PNG). 

*Note: Video formats like WebM and MP4 may be supported in future updates.*
