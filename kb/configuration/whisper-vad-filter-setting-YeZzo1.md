# Whisper VAD Filter Setting

> Enable VAD filter in Whisper settings to improve transcription.

## Answer

Enabling the **VAD filter** (Voice Activity Detection) in Whisper settings can help with transcription issues. This setting is typically enabled by default but should be checked if experiencing problems.
