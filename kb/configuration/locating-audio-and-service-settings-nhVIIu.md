# Locating Audio and Service Settings

> Access audio input and service configuration via the cogwheel icon in the Voxta UI.

## Answer

To configure active services and audio input settings (such as NAudio on Windows), navigate to the **Settings** menu by clicking the cogwheel icon located in the top right corner of the Voxta UI.
