# Florence-2 Base Model HuggingFace Repository

> Location of the official Florence-2 Base model on HuggingFace.

## Answer

The Florence-2 Base model is available on HuggingFace at `microsoft/Florence-2-base`. This model offers a balance of accuracy and speed and can be configured in Voxta by specifying the path `hf:microsoft/Florence-2-base`.
