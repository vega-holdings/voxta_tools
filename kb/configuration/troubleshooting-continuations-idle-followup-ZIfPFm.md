# Troubleshooting continuations_idle_followup

> Explanation of how mode and triggers affect the continuations_idle_followup feature.

## Answer

The `continuations_idle_followup` feature behavior depends on the active mode and triggers:

*   **Companion/Normal Mode:** It generally requires an action trigger. Keywords like "continue" or "go on" usually trigger it.
*   **Story Telling Mode:** It is designed to trigger automatically in this mode.
*   **Configuration:** Ensure the specific settings within the module are configured correctly.
