# ComfyUI Local Image Generation

> ComfyUI runs image generation locally on the user's hardware.

## Answer

Yes, ComfyUI is a local service. When you download and use models like `richyrichMIXIXL` within ComfyUI, the image generation process runs entirely on your local hardware.
