# Synchronizing Context Limits with KoboldAI

> Match the Voxta context limit to the KoboldAI backend limit.

## Answer

When configuring Voxta with KoboldAI, it is best practice to ensure the context limit in Voxta matches the limit set in the KoboldAI backend. For example, if KoboldAI is set to 4096 tokens, the Voxta configuration should also be set to 4096.
