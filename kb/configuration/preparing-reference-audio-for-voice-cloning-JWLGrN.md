# Preparing Reference Audio for Voice Cloning

> Recommended tools and format (22kHz mono, 9s) for preparing voice cloning reference audio.

## Answer

When preparing reference audio files for voice cloning or local TTS models, the following workflow is recommended to ensure quality:

*   **Tools:** Use **Audacity** to edit and format the audio, and **Adobe Podcast** to enhance clarity and remove noise.
*   **Format:** A suggested specification is **22 kHz mono** with a duration of approximately **9 seconds**.
*   **Local Options:** F5-TTS is noted as a viable local provider for those with sufficient hardware capabilities.
