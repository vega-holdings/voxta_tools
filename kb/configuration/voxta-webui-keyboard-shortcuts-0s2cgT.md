# Voxta WebUI Keyboard Shortcuts

> List of native keyboard shortcuts for Voxta WebUI including sidebar toggles, chat controls, and editor navigation.

## Answer

The following keyboard shortcuts are available in the Voxta WebUI for navigation, chat control, and editing:

### Chat & Interface
* **Ctrl + S**: Blurs the focused input field (prevents the browser's 'Save page' dialog).
* **Shift + Alt + M**: Toggles the left sidebar.
* **Shift + Alt + I**: Forces the right sidebar into Inspector view.
* **Shift + Alt + S**: Focuses the Services tab.
* **Shift + Alt + H**: Toggles hidden characters/items.
* **Shift + Alt + R**: Retries the current response.
* **Shift + Alt + T**: Speaks the last assistant message (TTS).
* **Shift + Alt + K**: Skips the current reply.
* **Shift + Alt + X**: Navigates back to the Home screen (`/`).
* **Shift + Alt + 1** through **5**: Cycles chat display layouts (Talk, Portrait, Chat, Assistant, Debug).

### Editors (Scenarios & Resources)
* **PageUp / PageDown**: Moves selection to the previous or next event/action item in the editor.
* **Ctrl + S**: Saves the current resource (works for Memory Books, Characters, Scenarios, Packages, Collections, and Configuration sets).
