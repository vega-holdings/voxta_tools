# Optimizing Character Cards for Specific Languages

> Translate the full character card into the target language for better LLM performance.

## Answer

To achieve the best results when interacting with a character in a specific language, it is recommended to translate the entire character card (including definitions, descriptions, and scenarios) into that preferred language.
