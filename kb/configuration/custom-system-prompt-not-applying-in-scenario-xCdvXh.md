# Custom System Prompt Not Applying in Scenario

> Fix for custom system prompts not applying by using the TextGen section in Module Manager.

## Answer

If a custom prompt defined in a scenario is not appearing in diagnostics, try configuring it in the **Module Manager** under the **TextGen** section. Users have reported that the scenario-level prompt might not take priority automatically without adjusting settings in the TextGen module.
