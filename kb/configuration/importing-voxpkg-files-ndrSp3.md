# Importing .voxpkg files

> Use the Voxta UI Import function for .voxpkg files rather than manual placement.

## Answer

You do not need to place the `.voxpkg` file in a specific system folder manually. You can save the file anywhere on your computer and use the **Import** function within the Voxta user interface to load it.
