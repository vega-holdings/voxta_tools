# Default Top A Value for ExLlamaV2

> Identifies the default Top A value as 0 (disabled) and its valid range.

## Answer

If the Top A field is empty, the default value is typically `0`, which effectively disables Top A sampling. The valid range for this parameter is `0.0` to `1.0`.
