# Fixing Idle.fbx Animation Issues

> Place Idle.fbx in a sub-folder named VRM to fix loading issues.

## Answer

To ensure `Idle.fbx` loads correctly, place the file inside a sub-folder named `VRM`. This specific folder structure is required for the animation to be recognized.
