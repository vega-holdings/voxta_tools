# Configuring Voxta Cloud Settings

> Voxta Cloud is configured under Manage Modules > Voxta Cloud Config.

## Answer

To configure the Voxta Cloud connection, navigate to **Manage Modules** and select **Voxta Cloud Config**.

This location is specific to the Voxta Cloud module; other third-party services (like ElevenLabs, Azure, or NovelAI) are typically configured via the **Services** menu.
