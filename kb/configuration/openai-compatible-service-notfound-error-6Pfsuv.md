# OpenAI Compatible Service NotFound Error

> Fix for OpenAI Compatible NotFound error caused by trailing slashes in the URL.

## Answer

This error is often caused by having a trailing slash (`/`) at the end of the API URL in the OpenAI Compatible service configuration. Removing the trailing slash from the URL usually resolves the issue.
