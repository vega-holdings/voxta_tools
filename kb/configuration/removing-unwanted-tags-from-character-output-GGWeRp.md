# Removing unwanted tags from Character output

> Use Text Replacements in Voxta Utilities to filter out unwanted tags from LLM output.

## Answer

You can use the **Text Replacements** feature located in **Voxta Utilities**. 

Configure **Character Reply Replacements** or **Speech Replacements** to catch specific patterns (such as `<pause>`) and replace them with an empty string or alternative text.
