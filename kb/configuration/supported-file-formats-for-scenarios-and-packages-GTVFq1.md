# Supported File Formats for Scenarios and Packages

> Scenarios can be loaded from PNG images or JSON, and packages support .voxpkg and .json extensions.

## Answer

Voxta supports multiple file formats for scenarios and packages:

*   **Scenarios:** Can be stored in `.png` images (embedding metadata, similar to character cards) and other formats.
*   **Packages:** Valid extensions include `.voxpkg` and `.json`.
