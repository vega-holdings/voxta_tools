# Purpose and Configuration of discord.dll

> discord.dll is an experimental feature configured via appsettings.json.

## Answer

The `discord.dll` module is associated with **experimental features** in Voxta. You can find its configuration settings in the `appsettings.json` file located in the Voxta root directory, specifically under the `ExperimentalFeatures` section.
