# Aetherroom Scenario Compatibility

> Aetherroom .scenario and world.info files are not compatible with Voxta.

## Answer

No, Voxta does not currently support importing `.scenario` or `world.info` files from Aetherroom. These file formats are not compatible with Voxta's import system.
