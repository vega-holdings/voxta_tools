# Profile Settings
The Profile Settings is the first screen you encounter. It allows you to customize how the AI will interact with you.

![Terms](https://doc.voxta.ai/images/uploads/snag-0243.png)
## Configuration
This section allows you to set your profile details:
  * **Profile Name (Required):** This is a required field. Enter the name that the AI character will call you. It is used in the **{{user}}** field.
  * **Profile Description:** Provide a description that will be used in the **{{user_description}}** field.


![Configuration](https://doc.voxta.ai/images/uploads/snag-0244.png)
## General Options
Customize your experience further with these options:
![Options](https://doc.voxta.ai/images/uploads/snag-0252.png)
  * **Prevent Speech Interruptions:**
When enabled, this feature stops the recording of your voice while the AI character is speaking. This ensures that the character can complete its statements without being interrupted by your subsequent voice inputs.
When disabled: Disabling this feature (unchecking the checkbox) allows for more natural, fluid conversations with your AI character. This means you can interrupt the AI character mid-sentence, much like you would in a natural conversation with a human. This is especially useful if you want to stop the character from completing a lengthy explanation or take the conversation in a different direction.
Recommendation: Itâ€™s advisable to disable this option if youâ€™re using headphones, as it allows for a seamless conversational experience without the risk of audio feedback.
  * **Hide NSFW Options and Characters:** Hide characters that are NSFW.


## Assistant
Define your default assistant character:
![Assistant](https://doc.voxta.ai/images/uploads/snag-0246.png)
  * **Assistant (Default Character):** Choose your personal assistant from a dropdown menu. The available options are: Voxta, Catherine, George. This character will be loaded by default when you launch Voxta, and will have the ability to run functions. This is your personal assistant, and will have more privileges such as executing plugins that you register.
