# Deepgram
Deepgram is a speech-to-text platform (STT) that provides real-time transcription of audio files. It is designed to help developers build intelligent voice applications with ease. Deepgram‚Äôs speech-to-text API uses advanced language models to transcribe audio files with high accuracy and speed.
<https://deepgram.com/>[](https://deepgram.com/)
## Acquiring an API Key from Deepgram
  * If you don‚Äôt have a [Deepgram account](https://deepgram.com/pricing), create one and select an appropriate subscription plan.
  * Log in to your Deepgram account.
  * Open the Dashboard from the main menu.
  * Locate and click on the ‚ÄúCreate API Key‚Äù button.
  * Give your API key a name, set the permissions, and configure the expiration settings as required.
  * Click to generate the API key. Note that it will only be displayed once, so make sure to copy it immediately.
  * Securely store the copied API key, as it is not recoverable once you navigate away from the page.


![DeepgramAPI](https://doc.voxta.ai/images/uploads/snag-0265.png)
## Integrate the Deepgram API Key into Voxta
  * Open the Voxta application on your device.
  * Navigate to the Services menu.
  * Click on the ‚Äú+ Add Services‚Äù option.
  * Scroll to find Deepgram in the list of available services, then click the ‚Äú+ Add‚Äù button next to it.
  * A designated input field will appear for the Deepgram API key. Paste your API key into this field.
  * Click ‚ÄúSave‚Äù to complete the integration.
