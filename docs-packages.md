# Packages
Packages are a way to bundle your content together for easy sharing and distribution. They can include scenarios, characters, memory books, and other assets.
You can create resources directly from the Package screen, or you can open existing resources and move them in your package. Keep in mind, a resource can only be in one package at a time, and deleting a package will delete everything inside.
It is recommended to export to `.voxpkg` format to minimize confusion, or `.png` for bundling with [Virt-A-Mate](https://doc.voxta.ai/vam/var-packages/).
A few useful notes:
  * When you assign a character or a scenario to a package, the packageâ€™s thumbnail and explicit content field will only be set when you open it again from the Studio menu. From there, the package will automatically be updated, and you can simply save it again.
  * Packages are required to export character and scenario assets.
