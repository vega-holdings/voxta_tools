ElevenLabs is a voice AI research & deployment company with a mission to make content universally accessible in any language & voice.
[https://elevenlabs.io](https://elevenlabs.io/?from=partnercobb805)
## Acquiring an API Key from 11ElevenLabs
### Step 1: Choose and Purchase a Subscription
  * Navigate to the [11ElevenLabs subscription page](https://elevenlabs.io/?from=partnercobb805).
  * Browse through the available plans and select one that suits your needs. Note: The $5 Starter subscription is usually sufficient for obtaining an API key.
  * Complete the purchase process. If you haven‚Äôt already signed up for an 11ElevenLabs account, you‚Äôll be prompted to do so.

  * Navigate to the ‚ÄúAccount‚Äù tab within ‚ÄúProfile Settings.‚Äù
  * Your API key will be displayed here, hidden by bullets for security reasons. To copy it, click on the field containing the API key and copy it to your clipboard.


![11LabsAPI](https://doc.voxta.ai/images/uploads/311.png)
### Step 3: Integrate the API Key into Voxta
