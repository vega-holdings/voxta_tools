# Chat
The **chat** page in Voxta is a dynamic space where users can have conversations with an AI character. You can either type messages or speak directly into a microphone. The AI listens, understands, and responds just like a real person would. What‚Äôs more, it uses emoticons that change based on the conversation‚Äôs tone and context, adding another layer of interaction. This page is designed to provide a seamless and interactive experience, simulating a real-life conversation with a virtual character.
![](https://doc.voxta.ai/images/uploads/_video_tutorial-voxta_doc-chat.gif)
## Chat view
**Chat view** mode is optimized for users who prefer typing to communicate with the AI. This mode offers additional functionalities like regenerating AI responses, editing messages, and deleting them if needed. These extra features provide more control over the conversation flow, making the chat experience more comprehensive and user-friendly for those who want to engage in more detailed and advanced interactions with the AI
![](https://doc.voxta.ai/images/uploads/_video_tutorial-voxta_doc-chat_view.gif)
## Scenario Chat with multiple Characters
When in a Chat with multiple Characters, Full Chat mode and Quick Access Menu will give a better view for knowing who is talking and seing Chat history.
![](https://doc.voxta.ai/images/uploads/_video_tutorial-voxta_doc-chat_multi_characters.gif)
## Chat inspector
The Chat Inspector is a tool within chat view mode that allows you to see and manage the internal workings of a chat with an AI. It shows the dialogue, the AI‚Äôs current context, functions that can be triggered, any retained memory, and other tools. To open the Chat Inspector, you need to click the ‚Äú`Show chat inspector`‚Äù button.
![](https://doc.voxta.ai/images/uploads/_video_tutorial-voxta_doc-chat_inspector.gif)
### Context
Context field in Voxta‚Äôs Chat Inspector is essentially a tool to direct the AI‚Äôs responses. You can use it to set up a situation or background information, like ‚Äú`character has a severe headache`‚Äù With this info, the AI will shape its replies accordingly. So, if you later ask the AI how the character is feeling, it will remember to mention the headache. This feature is useful for keeping the AI on track with a specific storyline or avoiding certain topics.
### Functions
These are AI-driven actions that can be invoked to enhance interaction, such as triggering different emotive responses or initiating specific tasks within the chat.
### Memory
This panel is where the AI stores any pertinent information that needs to be recalled throughout the interaction, ensuring a seamless conversation flow.
### Tools
Lists additional functionalities that can be employed to augment the AI‚Äôs capabilities in the chat. The Tools section in Voxta‚Äôs Chat Inspector includes an ‚Äú`Auto Reply`‚Äù feature, which automates the AI‚Äôs text generation for continuous conversation. This is particularly useful in storytelling scenarios where you want the AI to keep generating the story without needing a prompt from the user each time. By toggling on the auto reply, the AI will keep the narrative flowing on its own.
### Advanced
Contains technical details like session ID and chat ID, which are used to identify and manage the chat session uniquely.
### Shortcuts
Shortcut | Description  
---|---  
`shift+alt+m` | Toggles the left panel.  
`shift+alt+i` | Opens the Chat Inspector.  
`shift+alt+s` | Opens the Services tab in the right panel.  
`shift+alt+h` | Toggles the display of hidden messages.  
`shift+alt+r` | Re-generates or retries the last message.  
`shift+alt+x` | Stops the chat and navigates to the home page.  
`shift+alt+k` | Skips the audio of the current message.
