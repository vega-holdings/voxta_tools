### Requirements

  * A subscription to any of the [Voxtaâ€™s Patreon Tier](https://www.patreon.com/voxta/membership).


## About Patreon Tiers
Different Patreon Tier will reward you with various perks.  
To access those perks, linking a Discord and a Patreon account is needed.\
Patreon Tier subscription operate on a monthly payment basis.  
A subscription can be upgraded, downgraded, or canceled at any time.  
Once a Patreon Tier is unlocked, the associated perks also become available, with no option for a refund.

## Voxta Cloud credits
Voxta Cloud is an easy-to-install, all-in-one solution for using Voxta with online services.  
Voxta Cloud uses a credit system to monitor usage (amount of generated tokens).  
The quantity of Voxta Cloud credits received depends on the selected Patreon Tier.
Billing period is set to restart on the first day of each month.  
If still subscribed on renewal day, Voxta Cloud credits are reset to the full amount for that tier, regardless of any remaining credits.
If depleted, Voxta Cloud credits can be replenished by upgrading to a higher Patreon Tier.  
Note that Patreon only charges the difference between Tiers.  
As a result only a proportionate amount of Voxta Cloud credits will be added.
Check out [Voxta Cloud](https://doc.voxta.ai/docs/voxta-cloud) for more.
