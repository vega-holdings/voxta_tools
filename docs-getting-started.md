# Getting Started
Here is an overview of what is needed to get started with Voxta Server and Virt-A-Mate.
### Requirements
In order to access Voxta you will need the following:
  * A [Discord](https://discord.gg/voxta) account.
  * A [Patreon](https://www.patreon.com) account.
  * Both accounts to be [linked](https://support.patreon.com/hc/en-us/articles/212052266-Getting-Discord-access).
  * A subscription to any of the [Voxta Patreon Tier](https://www.patreon.com/voxta/membership).


Check out [Patreon](https://doc.voxta.ai/docs/patreon/) for more.
In order to use Voxta you will need the following:
  * A computer with Windows or Linux.
  * If using local free services for Voxta, a powerful GPU is needed.
  * If using online paid services for Voxta, an internet connection is needed.


Check out [Services](https://doc.voxta.ai/docs/services/) for more.
### Voxta Server
Voxta Server allows to run an AI powered companion using voice chat or text. Check out [Install the server](https://doc.voxta.ai/docs/install-server/) for a step by step guide to download and install.
### Virt-a-Mate
Virt-a-Mate (aka VAM) is an adult sandbox to create, modify and interact in real time with a 3D simulated model, either on Desktop or in VR.  
Virt-a-Mate is available separately on [MeshedVRâ€™s Patreon](https://www.patreon.com/meshedvr).  
Learn more about [Virt-a-Mate](https://doc.voxta.ai/vam/what-is-virt-a-mate/) and check out how to install [Voxta Virt-a-Mate Plugin](https://doc.voxta.ai/vam/installing-the-plugin/)  
Note that a subscription to [Voxta Patreon Companion Tier](https://www.patreon.com/voxta/membership) or higher is needed to access on [Discord](https://discord.gg/voxta), Voxta Virt-a-Mate Plugin as well as Voxta Scenes, Characters and Scenarios.
Check out [Patreon](https://doc.voxta.ai/docs/patreon/) for more.
